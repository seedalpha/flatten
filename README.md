# flatten

Flatten nested arrays

### Installation
    
    $ npm install seed-flatten --save

### Usage

```javascript
var flatten = require('seed-flatten');

flatten([1,2,3,[4,[[5]]],6]); // [1,2,3,4,5,6]

```
### Development
    
    $ git clone git@github.com:seedalpha/flatten.git
    $ cd flatten
    $ npm install
    $ npm test
